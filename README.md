# Programming for Everybody (Getting Started with Python)

This repository contains notes, exercises, and practice code for the course  
**Programming for Everybody (Getting Started with Python)** by Charles Severance.

It is part of the broader Python for Everybody learning track and is designed for complete beginners with no prior programming experience.

---

## 📘 Course Overview

**Programming for Everybody (Getting Started with Python)** introduces the basics of programming using the Python language. The course focuses on:

- Understanding fundamental programming concepts
- Writing simple Python programs
- Working with data, files, and web information
- Building problem-solving skills using code

This course is widely used as an entry point into programming and is known for its clarity and beginner-friendly approach.

## 🧠 What You Will Learn

By completing this course, you will be able to:

- Write and execute Python programs
- Use variables, expressions, and statements
- Work with conditional logic (`if`, `else`)
- Use loops (`for`, `while`)
- Define and use functions
- Handle strings and basic data structures
- Read and write files
- Work with web data and APIs (introductory level)
- Debug simple programs
